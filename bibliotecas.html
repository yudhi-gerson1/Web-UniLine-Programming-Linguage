Manual Oficial de Bibliotecas — UniLine v1.3

1. Intermediário (ww intermediary)
ACESSAR INTERMEDIARY
fn
Cria funções reutilizáveis. UniLine usa "uma linha" separada por x.
fn printarOla: x show("Olá") x call(printarOla)
fn printarNumero[numero]: x show(numero) x call(printarNumero[1])
call()
Executa uma função criada com fn.
call(printarOla)
opinion
Cria comentários de uma linha.
opinion: "Eu realmente amo programar!"
opinions / stopOpinion
Cria comentários de várias linhas. stopOpinion encerra os comentários.
opinions: "UniLine" x "Python" stopOpinion
crescent()
Coloca números em ordem crescente.
crescent(1, 9, 4, 5, 2) Saída: 1, 2, 4, 5, 9
decrescent()
Coloca números em ordem decrescente.
decrescent(1, 9, 4, 5, 2) Saída: 9, 5, 4, 2, 1
choose()
Escolhe itens aleatoriamente.
choose(1, 2, 3, 4, 5)
choose(1, 2, 3, 4, 5)[2]
wait() / waitms()
Pausa o terminal por segundos ou milissegundos.
wait(5) x show("5 segundos!")
waitms(5000) x show("5 segundos = 5000 milissegundos!")
2. Manipulação de Strings (ww strman)
ACESSAR STRMAN
cesar()
Aplica a cifra de Júlio César.
cesar("abcdef")[1] Saída: bcdefg
cesar("abcdef")[13] Saída: nopqrs
replace()
Substitui textos.
mensagem = "Oi, Yudhi" replace("Oi", "Olá", of mensagem) Saída: Olá, Yudhi
lower()
Transforma o texto em minúsculo.
lower("OLÁ!")
upper()
Transforma o texto em maiúsculo.
upper("olá!")
transform()
Extrai partes específicas de textos. <x> pega strings. <num> transforma em número. <bool> transforma em booleano. <char> pega apenas a inicial.
texto = "Olá, Yudhi" transform("Olá, ", "") Saída: Yudhi
3. Arrays (ww arrays)
ACESSAR ARRAYS
Variáveis de Lista
Cria listas.
afazeres = ["Fazer bolo", "Comprar farinha", "Colocar 3 ovos"]
fibonacci = [1, 1, 2, 3, 5, 8, 13, 21]
position()
Mostra a posição específica da lista.
impares = [1, 3, 5, 7, 9] position(3, impares) Saída: 5
total()
Soma todos os números da lista.
total([1, 2, 3]) Saída: 6
read()
Lê todos os itens do array separados por linha.
contar = [1, 2, 3] read(contar, 5)
divisibility()
Mostra apenas números divisíveis.
divisibility([1, 2, 3, 6, 78], 3) Saída: 3, 6, 78
show([array])
Arrays podem ser mostrados diretamente.
show([1, 2, 3]) Saída: 1, 2, 3
4. Booleanos e Pseudoaleatórios
verifyBool()
Compara booleanos usando operações.
intBool()
Transforma palavras booleanas em números.
kwBool()
Transforma números em palavras booleanas.
randomint()
Sorteia números inteiros.
randomint(1, 10)
randomdb()
Sorteia números decimais.
randomdb(1, 10)
randomstr()
Gera strings aleatórias.
randomstr(5)
5. Banco de Dados (ww files)
ACESSAR FILES
files
Sistema de memória permanente. Salva números, textos e códigos.
save()
Salva permanentemente arquivos.
save("Olá" as texto.txt) save("10" as pontos.num) save("show('Oi')" as script.uni)
show()
Mostra o conteúdo salvo do arquivo.
show(texto.txt)
forget()
Remove arquivos da memória permanente.
forget(texto.txt)
showMemory()
Mostra todos os arquivos salvos.
showMemory()
.uni
Arquivos de código UniLine.
.txt
Arquivos de texto.
.num
Arquivos numéricos.
.frac
Arquivos de frações mistas.
1int2db3
calculate()
Realiza operações matemáticas. Suporta +, -, *, /, % e elev.
calculate(a.num + b.num) calculate(a.num elev b.num)
concat()
Concatena textos ou soma números.
concat(nome.txt + sobrenome.txt)
6. Matemática Avançada (ww math)
ACESSAR MATH
Frações Mistas
int representa a parte inteira. db representa a parte fracionária.
1int2db3 2db3
mod
Mostra o resto da divisão.
calculate(9 mod 5) Saída: 4
d()[x]
Converte frações em decimal.
d(1int2db3)[15] Saída: 1.666666666666666
root()
Calcula raízes matemáticas.
root(2)[2, 3] Saída: 1.414
binary()
Converte números para binário.
binary(9) Saída: 1001
Tipo Binário
O b identifica valores binários. show() converte para decimal. binary() mantém binário.
binario = 1001b show(binario) Saída: 9 binary(binario) Saída: 1001
factorial()
Multiplica todos os números de 1 até o valor informado.
factorial(5) Saída: 120
