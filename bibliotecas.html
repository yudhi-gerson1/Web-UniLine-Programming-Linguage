Manual de Bibliotecas UniLine v1.2

ww intermediary (Intermediário)
Abrir Transpilador
1 - fn / call (Funções) Cria blocos de código reutilizáveis. Use fn para definir e call para executar.
fn printarOla: x show("Olá") x call(printarOla)
2 - opinion / opinions (Comentários) Notas para o programador. "opinion" para uma linha e "opinions/stopOpinion" para várias.
opinions: "UniLine" x "Python" stopOpinion
3 - crescent / decrescent Organiza números automaticamente. Imprime sozinho se não for guardado em variável.
crescent(1, 9, 4, 5, 2) x opinion: "Saída: 1, 2, 4, 5, 9"
4 - choose Sorteia um ou mais itens de uma lista aleatoriamente.
choose(1, 2, 3)[2] x opinion: "Escolhe 2 números da lista"
5 - wait / waitms Pausa o programa por segundos (wait) ou milissegundos (waitms).
wait(5) x show("Passaram 5 segundos")
ww strman (Manipulação de Strings)
Abrir Transpilador
I - cesar("string")[nível] Criptografa o texto usando a Cifra de César (deslocamento alfabético).
cesar("abcdef")[1] x opinion: "Saída: bcdefg"
II - replace("alvo", "novo", of contexto) Localiza um texto e o substitui por outro.
replace("Oi", "Olá", of "Oi, Yudhi")
III - lower() / upper() Transforma o texto em minúsculo ou MAIÚSCULO.
lower("OI!") x opinion: "oi!"
IV - transform() (Regex UniLine) Extrai dados específicos usando: <x>(texto), <num>(número), <bool>(lógico).
transform("Olá, <x>", texto) x opinion: "Extrai o nome"
ww arrays (Listas)
Abrir Transpilador
I - Variável de Lista Cria listas entre colchetes. Lembre-se: O primeiro item é o [1]!
lista = ["Fazer bolo", "Comprar ovos"]
II - position(n, variável) Mostra o valor na enésima posição.
position(3, [1, 3, 5]) x opinion: "Saída: 5"
III - total(variável) Soma todos os números de uma lista matemática.
total([1, 2, 3]) x opinion: "Saída: 6"
IV - read(variável, segundos) Lê a lista inteira, linha por linha, no tempo determinado.
read(lista, 5)
V - divisibility(lista, divisor) Filtra a lista mostrando apenas números divisíveis pelo divisor.
divisibility([1, 2, 3, 6], 3) x opinion: "Saída: 3, 6"
Nativas (Boolean & Random)
Boolean: verifyBool / intBool / kwBool Compara e converte estados yes/no em números ou palavras.
Random: randomint / randomdb / randomstr Gera números inteiros, decimais ou strings de letras aleatórias.
