ACESSAR TRANSPILADOR

Palavras-Chave Base

show()

Comando de saída. Projeta dados na interface.
Exemplo: show("Olá Mundo")

nome = valor

Atribuição de variável. Guarda informações no sistema.
Exemplo: pontos = 100

waitkb()

Wait KeyBoard. Pausa a linha e espera o usuário digitar.
Exemplo: nome = waitkb()

if(condicao)

A estrutura de decisão. Avalia se deve ou não executar o próximo passo.
Exemplo: if(check == yes)

elif(condicao)

Condição intermediária. Checa uma nova possibilidade caso o if falhe.
Exemplo: elif(check == no)

else(acao)

Caminho de falha. Executa caso nenhuma condição anterior seja real.
Exemplo: else(show("Erro"))

then(acao)

O gatilho da ação. É obrigatório após if/elif para dizer o que fazer.
Exemplo: if(y) x then(show("ok"))

verify(comparacao)

O motor de lógica. Compara dois valores e retorna obrigatoriamente yes ou no.
Exemplo: verify(10 == 10)

yes / no

Valores Booleanos. yes representa Verdade e no representa Falso.
Exemplo: resultado = yes

wdef

Will Define. Indica que a variável está vazia, mas será definida depois.
Exemplo: vida = wdef

calculate(conta)

Resolve operações matemáticas.
Exemplo: calculate(50 + 20)

elev

Operador de potência.
Exemplo: 10 elev 2 (Gera 100)

x

O divisor de comandos. Substitui o Enter.
Exemplo: comando1 x comando2

Controle de Fluxo e Loops

for(i=1, i++ until 5)

Loop contado com incremento.
Exemplo: for(i=1, i++ until 3): show(i)

while(1)

Loop de condição contínua.
Exemplo: while(ativo == yes): show("Rodando")

repeat(n)

Repete uma ação um número exato de vezes.
Exemplo: repeat(5): show("Olá")

Bibliotecas e Funções (ww)

ww (Work With)

Importa bibliotecas externas.
Exemplo: ww random

verifyBool()

Compara booleanos usando operações lógicas.
Exemplo: verifyBool(yes & no)

intBool() / kwBool()

Converte booleano para número e vice-versa.
Exemplo: intBool(yes) (Gera 1)

randomint(a, b)

Sorteia número inteiro entre a e b.
Exemplo: randomint(1, 10)

randomdb(a, b)

Sorteia número decimal.
Exemplo: randomdb(0, 1)

randomstr(a)

Gera string aleatória com 'a' letras.
Exemplo: randomstr(8)
